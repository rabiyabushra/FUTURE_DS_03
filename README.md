# College Event Feedback Analysis (Task 3)
This project analyzes student feedback collected after college events such as workshops, seminars, cultural programs, and competitions.
The goal is to understand student satisfaction using both numerical ratings and textual feedback through sentiment analysis.

## Tools & Technologies
- Python
- Google Colab
- pandas
- matplotlib & seaborn
- TextBlob (NLP)
- WordCloud

## Key Analysis Performed
- Rating distribution and average rating analysis
- Event-wise and event-type-wise performance
- Sentiment analysis of student feedback (Positive, Neutral, Negative)
- Word cloud of frequently used feedback terms
- Advanced visualizations such as heatmap and scatter plots

## Key Insights
- Workshops and hands-on events received the highest satisfaction.
- Seminars and tech talks showed scope for improvement.
- Most feedback sentiment was positive, indicating overall success of events.
- Students value engagement, practical exposure, and good organization.

## Recommendations
- Increase interactive and hands-on sessions.
- Improve engagement in seminars.
- Address logistical issues like scheduling and seating.
- Use feedback analytics regularly for event improvement.
